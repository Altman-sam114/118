# v1.34 / iPad 指针悬停基线

## 版本号

v1.34

## 版本分配依据

- 当前体验优化阶段已完成到 v1.33。
- v1.20 已提升 iPad regular sidebar 的可访问语义和 44pt 触控高度。
- v1.31-v1.33 已连续优化 Plan 的 iPad/付费/Mac 事实展示，但 Mac readiness 仍明确记录 window/sidebar、keyboard 和 pointer states 需要后续覆盖。
- 当前全仓未发现 `hoverEffect` 或等价 iPad pointer affordance；本轮选择一个小步 UI 体验目标：为 iPad sidebar rows 和共享 Sci-Fi 按钮补充系统 pointer hover affordance，不启用 Mac Catalyst，不声称 Mac 已支持。

## 背景

Local Diffusion 已经有 iPad regular 单层 `NavigationSplitView`，也有统一的 `SciFiPrimaryButtonStyle` 和 `SciFiSecondaryButtonStyle`。这些控件视觉上保持暗色科幻风格，但在 iPad trackpad/mouse 场景下没有显式 hover affordance。

本轮只补系统级指针悬停反馈，让 sidebar rows 和共享 Sci-Fi 按钮更符合 iPad 指针交互预期。它是 Mac readiness 中 pointer state 的 UI 基线，不是 Mac app 验收，也不是 Mac Catalyst 支持。

## 目标

- 在 `LocalDiffusion/Views/RootContentView.swift` 的 `SidebarSectionRow` 上增加明确 hit shape 和系统 hover affordance：
  - 保持现有 `Label`、44pt 最小高度、selected/not-selected accessibility value 和 workspace hint。
  - 建议增加 `.contentShape(RoundedRectangle(cornerRadius: 8))`。
  - 建议增加 `.hoverEffect(.highlight)`。
- 在 `LocalDiffusion/Views/Shared/ParameterEditor.swift` 的共享按钮样式中增加系统 hover affordance：
  - `SciFiPrimaryButtonStyle`
  - `SciFiSecondaryButtonStyle`
  - 保持现有 pressed opacity/shadow、disabled 颜色、圆角、暗色科幻视觉和可见 label 不变。
  - 建议在最终 shaped view 上增加 `.contentShape(RoundedRectangle(cornerRadius: 8))` 和 `.hoverEffect(.highlight, isEnabled: isEnabled)`。
- 同步 README、flow、flowchart、update_log。

## 非目标

- 不启用 Mac Catalyst，不修改 `SUPPORTED_PLATFORMS`、`SUPPORTS_MACCATALYST`、Xcode project、签名、分发或 scheme 设置。
- 不修改 native XCFramework、native bridge ABI、Release asset、CI workflow 或 Mac/Catalyst slice。
- 不新增 StoreKit import、product ID、价格、购买、恢复、receipt、subscription、entitlement persistence 或付费门禁。
- 不修改 v1.33 的 `Paid candidates` Availability 行、Entitlement Rules 或 Capability Matrix 付费事实。
- 不新增 visible keyboard shortcut 文案、Mac-only UI、菜单命令或 pointer 自定义动画。
- 不修改 Generate、Models、Gallery、Prompts 的业务行为、SwiftData schema、文件存储、native backend 或生成流程。

## 当前架构依据

- `SidebarSectionRow` 属于 Root iPad regular 导航 UI，当前只负责展示和辅助功能语义，不拥有业务状态。
- `SciFiPrimaryButtonStyle` 和 `SciFiSecondaryButtonStyle` 是共享 SwiftUI button styles，已被 Generate、Models、Gallery、Prompts 等页面复用。
- SwiftUI `hoverEffect` 是系统 pointer affordance，适合 iPadOS pointer 场景；本轮不做自定义 pointer animation。
- Mac readiness 仍只是 Plan 展示信息；本轮不能把 iPad pointer affordance 当作 Mac app 支持完成。
