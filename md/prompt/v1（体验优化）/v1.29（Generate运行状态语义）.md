# v1.29 / Generate 运行状态语义

## 版本号

v1.29

## 版本分配依据

- 当前体验优化阶段已完成到 v1.28。
- v1.15 已优化 Generate prompt 编辑器可读性，v1.13 已优化共享参数控件语义。
- Generate 页的运行门禁卡片、生成中进度、取消操作和结果预览仍缺少更具体的 VoiceOver 状态语义，尤其是 `GenerationGatePanel` 与 `Image(uiImage:)` 这种图文组合元素。

## 背景

Generate 页面是本地生成主入口。用户在此页面会看到：

- 后端 / 模型 / prompt 是否满足生成条件的门禁卡片。
- 生成中进度和取消按钮。
- 生成完成后的结果预览和 View in Gallery 操作。

本轮只改善这些运行状态和结果预览的辅助功能语义，不改变任何生成、保存、取消、跳转或数据层行为。

## 目标

- 为 `GenerationGatePanel` 增加组合 accessibility label/value/hint：
  - label 表达这是 generation status。
  - value 包含 gate title 和 message。
  - hint 根据当前 gate 状态提示下一步：后端离线、缺模型、缺 prompt 或 ready to render。
- 为生成中 `ProgressView` 增加组合 accessibility label/value/hint：
  - value 包含当前 stage 和百分比。
  - 百分比应从 `viewModel.progress.fraction` 派生并限制在 0...100。
- 为 Cancel 按钮增加明确 accessibility label/value/hint，保留 destructive role、禁用条件和现有 action。
- 为结果预览 image 增加 accessibility label/value/hint：
  - label 表达这是 generated image preview。
  - value 使用解码后的 `UIImage` 像素尺寸和 Gallery 保存状态。
  - 不用当前可编辑参数拼接结果描述，避免用户生成后又改参数导致朗读过期信息。
- 为 `View in Gallery` 按钮增加 ready/unavailable value 和明确 hint。

## 非目标

- 不修改 `GenerationViewModel`、`GenerationProgress`、`ImageGenerationBackend`、native bridge 或生成请求。
- 不修改生成门禁判断、`canGenerate`、`generationGate` 的业务条件、生成启动、取消、保存或 Gallery 跳转逻辑。
- 不修改 prompt editor、ParameterEditor、Model picker、SwiftData schema、文件存储、StoreKit、Mac Catalyst、Xcode project 或 CI workflow。
- 不新增 simulator screenshot、真机 VoiceOver 目检或真实 GGUF 生成验收。

## 当前架构依据

- `GenerationView` 属于 UI 展示层，只负责展示 ViewModel 状态、收集输入和调用现有 action。
- `GenerationViewModel` 负责生成流程、进度、取消和保存结果，本轮不得把展示语义下沉到 ViewModel。
- SwiftUI 可访问性要求：图标、标题、说明组成的状态卡片应合并为一个有意义的辅助功能元素；图片若不是装饰图，应提供明确 label/value。

3. 在 `GenerationGatePanel` 外层添加：
   - `.accessibilityElement(children: .ignore)`
   - `.accessibilityLabel("Generation status")`
   - `.accessibilityValue("\(gate.title). \(gate.message)")`
   - `.accessibilityHint(gate.accessibilityHint)`
4. 为生成中 `ProgressView` 添加辅助功能修饰，并在 `GenerationView` 中增加私有 helper：
   - `generationProgressAccessibilityValue`

   - `resultImageAccessibilityValue(for image: UIImage)`
   - value 应使用 `image.size * image.scale` 推导像素尺寸，并根据 `latestGeneratedImageID` 表达 Saved to Gallery / Gallery record unavailable。
7. 为 `View in Gallery` 按钮添加 ready/unavailable value 和打开已保存结果详情的 hint。
8. 更新文档：
   - `README.md`
   - `md/flow/flow.md`
   - `md/flow/flowchart.md`
   - `update_log.md`
