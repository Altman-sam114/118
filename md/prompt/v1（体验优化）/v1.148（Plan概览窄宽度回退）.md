# v1.148 / Plan 概览窄宽度回退

## 版本号

- 版本：`v1.148`
- 阶段：`v1（体验优化）`

## 版本分配依据

- 当前最新正式版本为 `v1.147 / SciFiMetric 窄宽度回退`。
- 人工总目标要求持续优化 Mac/iPad/UI/设计/付费功能。
- Plan note/status rows 已完成多种窄宽度回退；`planOverview` 顶部 icon + copy 在普通 Dynamic Type 下仍固定横排，是下一个小而可验收的 UI 韧性优化。

## 背景

`PlanView.planOverview` 在 accessibility Dynamic Type 下已纵向堆叠 `planOverviewIcon` + `planOverviewCopy`，但普通 Dynamic Type 固定 `HStack(alignment: .top, spacing: 12)`。在 iPad split / 窄 regular 宽度下可能挤压。本轮用 `ViewThatFits(in: .horizontal)` 对齐既有模式，不改变 Local plan / StoreKit / paid candidates 边界文案与 VoiceOver 语义。

## 目标

1. 只修改 `planOverview` 顶部 icon/copy 的普通 Dynamic Type 分支。
2. `ViewThatFits(in: .horizontal)` 优先横排 icon + copy，不足时回退纵向。
3. accessibility 纵向布局不变。
4. 保留 divider、purchase boundary 说明、panel 样式、accessibility label/value/hint 和所有 Plan 业务边界不变。

## 非目标

- 不修改 StoreKit、purchase UI、entitlement、paid gate、Mac Catalyst、status rows、note rows、native、SwiftData、workflow。
- 禁止本地 Xcode / Simulator / 真机构建测试。

## 实现步骤

```swift
if dynamicTypeSize.isAccessibilitySize {
    VStack(...) { planOverviewIcon; planOverviewCopy }
} else {
    ViewThatFits(in: .horizontal) {
        HStack(alignment: .top, spacing: 12) {
            planOverviewIcon
            planOverviewCopy
        }

        VStack(alignment: .leading, spacing: 12) {
            planOverviewIcon
            planOverviewCopy
        }
    }
}
```
