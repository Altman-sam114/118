## 版本号

- 版本：`v1.147`

## 版本分配依据

- 当前最新正式版本为 `v1.146 / Generate 控制台状态 Pills 窄宽度回退`。
- 人工总目标要求持续优化 Mac/iPad/UI/设计/付费功能。
- Generate 控制台 overview/pills 已完成窄宽度回退；共享 `SciFiMetric` 在普通 Dynamic Type 下仍固定横排 icon + copy，是下一个小而可验收的 UI 韧性优化。

## 背景

`SciFiMetric` 在 accessibility Dynamic Type 下已纵向堆叠 icon + copy，但普通 Dynamic Type 固定 `HStack(spacing: 10)`。在窄 adaptive grid cell 中可能挤压。本轮用 `ViewThatFits(in: .horizontal)` 对齐既有模式。

## 目标

1. 只修改 `SciFiMetric.body` 的普通 Dynamic Type 分支。
2. `ViewThatFits(in: .horizontal)` 优先横排 icon + copy，不足时回退纵向。
3. accessibility 纵向布局不变。
4. 保留 accessibility label/value、颜色、icon、title/value 文案不变。

## 非目标

- 不修改 consoleMetricColumns、ParameterEditor 其他控件、generation gate、native、StoreKit、Mac Catalyst、workflow。
- 禁止本地 Xcode / Simulator / 真机构建测试。

## 实现步骤

```swift
ViewThatFits(in: .horizontal) {
    HStack(spacing: 10) {
        metricIcon
        metricCopy
    }

    VStack(alignment: .leading, spacing: 8) {
        metricIcon
        metricCopy
    }
}
```
