# v1.41 / Gallery 排序菜单当前值语义

## 版本号

v1.41

## 版本分配依据

- 当前体验优化阶段已完成到 v1.40。
- v1.17、v1.35、v1.37 已持续改善 Gallery filter rail 和 image tile 的可读性、VoiceOver 摘要与 pointer hover affordance。
- Gallery 工具栏的 `Sort` Picker 当前只有通用 hint：`Changes the order of visible gallery images.`，没有明确暴露当前排序值。
- Gallery 支持 `Newest`、`Oldest`、`Model` 三种排序。VoiceOver 聚焦 Sort 菜单时，如果只听到通用菜单名称，用户需要打开菜单才知道当前排序状态。
- 本轮选择一个小步 iPad/Mac readiness UI 目标：为 Gallery Sort 菜单补充当前排序 VoiceOver value，不改变排序逻辑、筛选、导航、图库详情或文件状态行为。

## 背景

Local Diffusion 的 Gallery 是生成图片浏览、筛选和复用入口。用户可按最新、最旧或模型名排序可见图片。当前 `imageGridToolbar(refreshPlacement:)` 中的 Sort Picker 使用 `.pickerStyle(.menu)`，视觉和行为已符合工具栏菜单预期，但辅助功能语义没有把当前 sort state 独立表达出来。

本轮只给 Sort Picker 增加清晰的辅助功能 label/value，并用一个纯展示 helper 将 `GallerySort` case 转成更适合朗读的句子。排序数据源、`visibleImages` 排序实现、Picker options 和 toolbar placement 都保持不变。

## 目标

- 在 `LocalDiffusion/Views/Gallery/GalleryView.swift` 的 `imageGridToolbar(refreshPlacement:)` 中，为 Sort Picker 增加：
  - `accessibilityLabel("Gallery sort order")`
  - `accessibilityValue(...)`
- 新增一个只读展示 helper，例如 `sortAccessibilityValue`：
  - `.newest` -> `Newest images first`
  - `.oldest` -> `Oldest images first`
  - `.model` -> `Grouped by model name`
- 保持现有 `Picker("Sort", selection: $sort)`、`ForEach(GallerySort.allCases)`、`.pickerStyle(.menu)`、排序选项和 hint 不变。
- 同步 README、flow、flowchart、update_log，说明 Gallery Sort menu 获得当前排序值 VoiceOver semantics baseline。

## 非目标

- 不修改 `GallerySort` case、rawValue、排序选项文案或 `visibleImages` 排序实现。
- 不修改 Gallery filter rail、image grid、image tile、detail view、folder/tag 组织、delete/share/reuse/regenerate 或 file reconciliation 行为。
- 不修改 SwiftData schema、文件存储、native backend、Generate、Models、Prompts 或 Plan 业务行为。
- 不启用 Mac Catalyst，不修改 `SUPPORTED_PLATFORMS`、`SUPPORTS_MACCATALYST`、Xcode project、签名、分发或 scheme 设置。
- 不新增 StoreKit import、product ID、价格、购买、恢复、receipt、subscription、entitlement persistence 或付费门禁。
- 不新增 visible keyboard shortcut 文案、Mac-only UI、自定义 hover 动画或额外手势。

## 当前架构依据

- Gallery 属于 `LocalDiffusion/Views/Gallery/GalleryView.swift` 的 UI 展示层，只负责展示、输入、导航和用户反馈。
- `sort` 是 Gallery view 本地 UI state；`visibleImages` 基于它计算可见顺序。
- `imageGridToolbar(refreshPlacement:)` 是 Sort Picker 的最小作用点。
- 新增 helper 只把当前 sort state 转成辅助功能朗读文本，不参与业务排序或数据持久化。
