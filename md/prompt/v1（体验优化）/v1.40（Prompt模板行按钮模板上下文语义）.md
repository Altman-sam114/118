# v1.40 / Prompt 模板行按钮模板上下文语义

## 版本号

v1.40

## 版本分配依据

- 当前体验优化阶段已完成到 v1.39。
- v1.11 已让 Prompt template row 的 icon-only edit/load controls 具备可访问文本 label、44pt 命中区和 Dynamic Type 堆叠布局。
- v1.28、v1.38 已继续改善 Prompt Library 的空状态、搜索空状态和分类菜单上下文语义。
- Prompt Library 中每个模板行都可能显示重复的 `Edit Template` 和 `Load Template` icon-only 按钮；当前按钮通过 `Label` 暴露通用动作名，但没有把具体 `template.name` 编入 VoiceOver label。
- 本轮选择一个小步 iPad/Mac readiness UI 目标：为 Prompt 模板行操作按钮补充模板名上下文辅助功能语义，不改变模板编辑、加载、查询、排序或分类行为。

## 背景

Local Diffusion 的 Prompt Library 用于保存、编辑和加载生成提示词模板。用户可能有多个同名类别下的模板，每个 row 都显示相似的 edit/load controls。共享 `SciFiSecondaryButtonStyle` 已提供视觉样式、hit shape 和 pointer hover affordance，按钮本身也有 44pt 最小尺寸。

当前 `PromptTemplateRow.actions` 中的按钮只暴露通用动作名，例如 `Edit Template` 或 `Load Template`。在多模板列表中，这些重复 label 对 VoiceOver 和 Voice Control 不够具体。本轮只给现有按钮追加模板名上下文和简短 hint，保持视觉、布局和 action 不变。

## 目标

- 在 `LocalDiffusion/Views/Prompts/PromptLibraryView.swift` 的 `PromptTemplateRow.actions` 中，为 `Edit Template` 和 `Load Template` 按钮补充模板上下文辅助功能语义：
  - label 示例：`"Edit Template: \(template.name)"`。
  - label 示例：`"Load Template: \(template.name)"`。
  - hint 说明动作会编辑或加载当前模板。
- 保持现有 `Button(action:)`、`Label(title, systemImage:)`、`.labelStyle(.iconOnly)`、`SciFiSecondaryButtonStyle`、`.frame(minWidth: 44, minHeight: 44)`、按钮顺序和 closures 不变。
- 同步 README、flow、flowchart、update_log，说明 Prompt template row controls 获得模板名上下文 VoiceOver semantics baseline。

## 非目标

- 不修改模板查询、搜索、分组、排序、编辑表单、保存、加载、删除、分类 rename/clear 或参数复制行为。
- 不修改 SwiftData schema、文件存储、native backend、Generate、Gallery、Models 或 Plan 业务行为。
- 不修改 `SciFiSecondaryButtonStyle`、共享 button hover、整张 `PromptTemplateRow` 的点击行为、row hover 或卡片视觉。
- 不启用 Mac Catalyst，不修改 `SUPPORTED_PLATFORMS`、`SUPPORTS_MACCATALYST`、Xcode project、签名、分发或 scheme 设置。
- 不新增 StoreKit import、product ID、价格、购买、恢复、receipt、subscription、entitlement persistence 或付费门禁。
- 不新增 visible keyboard shortcut 文案、Mac-only UI、自定义 hover 动画或额外手势。

## 当前架构依据

- Prompt Library 属于 `LocalDiffusion/Views/Prompts/PromptLibraryView.swift` 的 UI 展示层，只负责展示、输入、导航和用户反馈。
- `PromptTemplateRow` 是单个模板 row 的局部 view；编辑和加载仍由外层 `PromptLibraryView` 传入的 closures 处理。
- `actions` 是模板行 edit/load controls 的最小作用点。
- 共享 `SciFiSecondaryButtonStyle` 已提供 pointer hover 和 hit shape，本轮不修改共享样式，避免扩大到无关按钮。
