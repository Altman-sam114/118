# v1.63 / Prompt 编辑器保存语义

## 版本号

v1.63

## 版本分配依据

- v1.62 已完成 Generate Run section accessibility 语义，并通过 `origin/main` 云端结果包验收。
- 本轮继续在 v1 体验优化阶段做小步、可验收的 Prompt Library 可访问性增强。

## 背景

Prompt Library 已有搜索、空状态、分类菜单、模板行操作、metric pills、prompt 编辑区和共享参数控件的可访问语义。`PromptTemplateEditor` 与 `PromptCategoryNameEditor` 的 toolbar Save/Cancel 按钮仍主要依赖可见标题和 disabled 状态，VoiceOver 对保存条件、名称必填和取消不保存的语义不够明确。

## 目标

- 为 `PromptTemplateEditor` 的 Cancel 按钮补充关闭并丢弃未保存编辑的提示。
- 为 `PromptTemplateEditor` 的 Save 按钮补充 ready/name required 状态和保存模板名称、分类、prompts、参数的提示。
- 为 `PromptCategoryNameEditor` 的 Cancel 按钮补充关闭并丢弃重命名编辑的提示。
- 为 `PromptCategoryNameEditor` 的 Save 按钮补充 ready/name required 状态和保存分类名称的提示。

## 非目标

- 不修改模板新增、编辑、保存、加载、删除、搜索、分类 rename/clear 的行为。
- 不修改 `PromptTemplate` SwiftData schema、`GenerationParameters`、文件存储、native backend、StoreKit、Mac Catalyst、Xcode project 或 GitHub Actions workflow。
- 不新增购买、订阅、entitlement、价格、product ID 或 Mac 平台开关。

## 当前架构依据

- UI 层只负责展示、输入、导航和用户反馈，位于 `LocalDiffusion/Views/**`。
- Prompt Library 通过 SwiftData 查询和 `GenerationViewModel.load(template:)` 复用模板，不直接访问 native bridge。
- 当前核心流程文档要求 Prompts 的模板编辑、加载控件和共享参数编辑控件在 accessibility Dynamic Type 下保持可读并提供清晰辅助功能语义。

## 实现步骤

1. 阅读 `PromptLibraryView.swift` 中 `PromptTemplateEditor` 与 `PromptCategoryNameEditor`。
2. 在 `PromptTemplateEditor` 内保留现有 toolbar placement、Save action、Cancel action 和 disabled 条件。
3. 给 template Cancel 增加稳定 label/value/hint，说明关闭编辑器且不保存当前更改。
4. 给 template Save 增加 stable label、ready/name required value 和保存内容范围 hint。
5. 在 `PromptCategoryNameEditor` 内同样保留现有 action 和 disabled 条件，只补 Cancel/Save label/value/hint。
6. 如需 helper，限制在当前 editor struct 内，避免新建跨文件抽象。
7. 同步 README、`md/flow/flow.md`、`md/flow/flowchart.md`、`update_log.md` 的当前 UI 语义事实。

## 验收标准

- Prompt template editor Save 在名称存在时向 VoiceOver 暴露 Ready 状态，并说明保存模板名称、分类、prompts 和参数。
- Prompt template editor Save 在名称为空时向 VoiceOver 暴露 Template name required。
- Prompt template editor Cancel 说明会关闭且不保存当前编辑。
- Category rename editor Save 在名称存在/缺失时分别暴露 Ready / Category name required。
- Category rename editor Cancel 说明会关闭且不保存重命名编辑。
