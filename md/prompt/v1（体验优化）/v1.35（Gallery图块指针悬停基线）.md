# v1.35 / Gallery 图块指针悬停基线

## 版本号

v1.35

## 版本分配依据

- 当前体验优化阶段已完成到 v1.34。
- v1.34 已为 iPad regular sidebar rows 和共享 Sci-Fi 主/次按钮建立系统 pointer hover affordance 基线。
- Gallery 图片图块是 iPad 宽屏和未来 Mac readiness 的核心浏览入口，目前仍是 plain `NavigationLink` 卡片，没有显式 hit shape 或系统 hover affordance。
- 本轮选择一个小步 UI 体验目标：为 Gallery 图片图块补充系统 pointer hover affordance，不改变 Gallery 导航、筛选、排序、详情、复用或文件行为。

## 背景

Local Diffusion 的 Gallery 已支持 compact 内部筛选 split、iPad Root detail 内嵌宽屏 filter rail、图片网格、详情导航、folder/tag 组织、分享、删除、复用和再生成。v1.12、v1.16、v1.17、v1.30 已经提升 Gallery 的 Dynamic Type 和 VoiceOver 语义。

当前 `ImageTile` 视觉上是暗色 Sci-Fi panel，并通过外层 `NavigationLink(value:)` 打开详情。iPad trackpad/mouse 场景下，图片图块缺少与 v1.34 已建立基线一致的系统 hover affordance。本轮只补交互可感知性，不引入自定义动画或新业务状态。

## 目标

- 在 `LocalDiffusion/Views/Gallery/GalleryView.swift` 的 `ImageTile` 上增加明确 hit shape 和系统 hover affordance：
  - 保持现有 `NavigationLink(value:)`、`.buttonStyle(.plain)`、图片渲染、尺寸徽章、prompt、metadata、`.sciFiPanel()` 和 accessibility summary。
  - 建议在 `.sciFiPanel()` 之后增加 `.contentShape(RoundedRectangle(cornerRadius: 8))`。
  - 建议增加 `.hoverEffect(.highlight)`。
- 同步 README、flow、flowchart、update_log，说明这是 Gallery 图块的 iPad pointer hover affordance baseline。

## 非目标

- 不启用 Mac Catalyst，不修改 `SUPPORTED_PLATFORMS`、`SUPPORTS_MACCATALYST`、Xcode project、签名、分发或 scheme 设置。
- 不修改 native XCFramework、native bridge ABI、Release asset、CI workflow 或 Mac/Catalyst slice。
- 不新增 StoreKit import、product ID、价格、购买、恢复、receipt、subscription、entitlement persistence 或付费门禁。
- 不修改 Plan paid candidates、Capability Matrix、Entitlement Rules、Availability 或 Mac readiness 的事实。
- 不修改 Gallery 的筛选、排序、详情导航、删除、分享、复用、再生成、folder/tag 保存、SwiftData schema、文件存储或图片 reconciliation 行为。
- 不新增 visible keyboard shortcut 文案、Mac-only UI、自定义 hover 动画或额外手势。

## 当前架构依据

- Gallery 属于 `LocalDiffusion/Views/Gallery/GalleryView.swift` 的 UI 展示层，只负责展示、输入、导航和用户反馈。
- `ImageTile` 是图片图块视觉和辅助功能摘要的局部 view；外层仍由 `NavigationLink(value:)` 负责导航。
- SwiftUI `hoverEffect(.highlight)` 是系统 pointer affordance，适合 iPadOS pointer 场景；本轮不做自定义 pointer animation。
- Gallery 数据仍来自 SwiftData `GeneratedImage` 和 `AppFileStore` 图片文件；本轮不能触碰数据层、文件层或推理层。
