# v1.135 / Prompt 模板指标窄宽度回退

## 版本号

- 版本：`v1.135`
- 阶段：`v1（体验优化）`

## 版本分配依据

- 当前最新正式版本为 `v1.134 / Plan 说明行无障碍回退`。
- 人工总目标要求持续优化 Mac/iPad/UI/设计/付费功能。
- 近期 v1.121-v1.134 已把 `ViewThatFits(in: .horizontal)` 窄宽度回退模式扩展到 Generate、Gallery、Plan 与共享参数控件；Prompt Library 模板行 metric pills 在普通 Dynamic Type 下仍固定横排，是下一个小而可验收的 UI 韧性优化。

## 背景

`PromptTemplateRow.metricPills` 在 accessibility Dynamic Type 下已经使用纵向 steps/sampler/size pill 布局，但普通 Dynamic Type 分支固定使用 `HStack(spacing: 8)`。模板 metric 在 iPad split、Stage Manager、窄 list 宽度或较长 sampler 名称下可能横向挤压。项目内已在 Gallery metadata、Plan status rows 与 ParameterEditor 中使用 `ViewThatFits(in: .horizontal)` 做窄宽度回退；本轮把同一模式补齐到 Prompt template metric pills，不改变模板业务、Edit/Load 控件或 VoiceOver 语义。

## 目标

1. 只修改 `PromptTemplateRow.metricPills` 的普通 Dynamic Type 分支。
2. 使用 `ViewThatFits(in: .horizontal)` 优先显示既有横向 steps/sampler/size pills。
3. 横向空间不足时回退到既有纵向 pill 布局。
4. 保持 accessibility Dynamic Type 现有纵向布局不变。
5. 保持 Edit/Load 按钮、row-level accessibility summary、metric pill label/value/hint、模板加载/编辑业务、颜色、图标、spacing 和 panel 样式不变。

## 非目标

- 不修改 `header`、`templateCopy`、`actions`、`templateAccessibilityValue` 或 row 外层 accessibility。
- 不修改 Prompt template editor、category editor、search、empty states 或 shared ParameterEditor。
- 不修改 pill 文案、颜色、system image、accessibility label/value/hint。
- 不修改 SwiftData schema、AppFileStore、GenerationViewModel、native backend、StoreKit、Mac Catalyst、Xcode project、workflow 或测试规范。
- 不做 simulator / Stage Manager / Split View / VoiceOver 实机目检；本轮通过 Swift parse 和云端 build 覆盖语法与构建风险。
- 禁止本地 Xcode / Simulator / 真机构建测试；完整构建交给 GitHub Actions。

## 当前架构依据

- `PromptLibraryView.swift` 位于 `LocalDiffusion/Views/Prompts/`，属于 SwiftUI UI 展示层。
- `PromptTemplateRow` 只展示 `PromptTemplate` 摘要并提供 Edit/Load 入口；metric pills 是纯展示。
- 本轮只改善普通字号下 metric pills 的布局回退，不改变数据或交互。

## 实现步骤

1. 阅读 `PromptTemplateRow.metricPills`、`stepsPill`、`samplerPill`、`sizePill`。
2. 保留 accessibility Dynamic Type 纵向分支。
3. 将普通 Dynamic Type 分支改为：

```swift
ViewThatFits(in: .horizontal) {
    HStack(spacing: 8) {
        stepsPill
        samplerPill
        sizePill
    }

    VStack(alignment: .leading, spacing: 8) {
        stepsPill
        samplerPill
        sizePill
    }
}
```

4. 不改 pill 内容、accessibility 或 header/actions。
5. 同步更新 README、`md/flow/flow.md`、`md/flow/flowchart.md` 和 `update_log.md`，说明 Prompt template metric pills 在窄宽度下可回退堆叠。
6. 检查 diff，确认只包含本轮相关源码、文档和本提示词。
