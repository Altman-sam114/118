# v1.143 / Generate Prompt 编辑器标题窄宽度回退

## 版本号

- 版本：`v1.143`

## 版本分配依据

- 当前最新正式版本为 `v1.142 / Prompt 编辑器标题窄宽度回退`。
- 人工总目标要求持续优化 Mac/iPad/UI/设计/付费功能。
- Prompt Library editor header 已完成窄宽度回退；Generate 页 `promptEditorHeader` 在普通 Dynamic Type 下仍固定横排 title + metadata，是下一个对称、可验收的 UI 韧性优化。

## 背景

`GenerationView.promptEditorHeader` 在 accessibility Dynamic Type 下已纵向堆叠 title + metadata，但普通 Dynamic Type 固定 `HStack { title; Spacer(); metadata }`。本轮用 `ViewThatFits(in: .horizontal)` 对齐既有模式。

## 目标

1. 只修改 `promptEditorHeader` 的普通 Dynamic Type 分支。
2. `ViewThatFits(in: .horizontal)` 优先横排，不足时回退纵向。
3. accessibility 纵向布局不变。
4. 保留 character count、clear button、prompt editing 行为不变。

## 非目标

- 不修改 ParameterEditor、generation gate、run/result、native、StoreKit、Mac Catalyst、workflow。
- 禁止本地 Xcode / Simulator / 真机构建测试。

## 实现步骤

```swift
ViewThatFits(in: .horizontal) {
    HStack {
        promptEditorTitle(title, accent: accent)
        Spacer()
        promptEditorMetadata(text: text, title: title)
    }

    VStack(alignment: .leading, spacing: 6) {
        promptEditorTitle(title, accent: accent)
        promptEditorMetadata(text: text, title: title)
    }
}
```
