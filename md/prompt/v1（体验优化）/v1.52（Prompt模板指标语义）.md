# v1.52 / Prompt 模板指标语义

## 版本号

v1.52

## 版本分配依据

- 当前体验优化阶段已完成到 v1.51。
- Prompt Library 已具备分类菜单上下文、空状态/搜索空状态语义、模板行 edit/load 模板名上下文、模板编辑器 prompt 字段语义和共享参数编辑器语义。
- `PromptTemplateRow` 的 steps、sampler、size 三个 metric pill 目前只显示短值，VoiceOver 缺少“这是模板去噪步数 / 采样算法 / 画布像素尺寸”的上下文。
- 本轮选择 Prompt 模板行指标的小步语义优化，不修改模板查询、分组、排序、保存、编辑、加载、参数值或 SwiftData 行为。

## 背景

Prompt Library 模板行会在标题和 prompt 摘要下展示三个参数指标：steps、sampler、size。视觉上用户能通过图标和位置理解这些值，但 VoiceOver 用户听到的短值不够明确，特别是数字 steps 和 `widthxheight` 尺寸。

本轮只为这三个 pill 补充辅助功能 label/value/hint。可视布局、Dynamic Type 下横向/纵向排列、模板行 panel、Edit/Load 按钮、模板编辑器和加载行为保持不变。

## 目标

- 在 `LocalDiffusion/Views/Prompts/PromptLibraryView.swift` 的 `PromptTemplateRow` 中：
  - 保留 `metricPills` 的现有布局和 Dynamic Type 分支。
  - 保留 `SciFiStatusPill` 的现有视觉样式、图标、颜色和 title。
  - 给 `stepsPill` 增加 VoiceOver label/value/hint，说明该值是模板的 denoising steps。
  - 给 `samplerPill` 增加 VoiceOver label/value/hint，说明该值是模板选择的 sampler algorithm。
  - 给 `sizePill` 增加 VoiceOver label/value/hint，说明该值是模板保存的 canvas pixel size。
- 同步 README、flow、flowchart、update_log，记录 Prompt 模板行指标会暴露 steps/sampler/size 上下文。

## 非目标

- 不修改 `PromptTemplate.matchesSearch(_:)`、`groupedTemplates`、`filteredTemplates`、分类 rename/clear、模板新增/编辑/删除或 load 行为。
- 不修改 `PromptTemplate` 或 `GenerationParameters` 的字段、默认值、范围、归一化、保存或加载逻辑。
- 不修改共享 `SciFiStatusPill`，避免影响 Generate、Models 或其它页面的现有状态语义。
- 不修改 SwiftData schema、文件存储、`GenerationViewModel`、native backend、Xcode project、CI workflow 或 Release asset。
- 不启用 Mac Catalyst，不新增 StoreKit import、product ID、购买、恢复、receipt、subscription、entitlement persistence 或付费门禁。
- 不新增 simulator VoiceOver 自动化、截图基线、Mac build、StoreKit 测试或真机 GGUF 生成验收。

## 当前架构依据

- `PromptLibraryView` 属于 `LocalDiffusion/Views/Prompts/PromptLibraryView.swift` 的 UI 展示层，只负责模板展示、分类操作、编辑入口和加载入口。
- `PromptTemplateRow` 已经拥有 template 上的 steps、samplerRawValue、width、height 显示值，本轮只为这些现有展示值增加辅助功能表达。
- 辅助功能 label/value/hint 属于 SwiftUI UI 表达，不应进入 SwiftData、文件存储、ViewModel 或 native bridge 边界。
- 共享 `SciFiStatusPill` 被多个页面复用；本轮只改 Prompt 模板行调用点，保持改动范围可控。
