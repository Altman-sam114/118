# v1.45 / 共享采样器菜单语义

## 版本号

v1.45

## 版本分配依据

- 当前体验优化阶段已完成到 v1.44。
- v1.42、v1.43、v1.44 已连续补充共享 `ParameterEditor` 中 Reset Defaults、Randomize Seed 和 Canvas Size preset 的 VoiceOver 语义。
- `ParameterEditor` 的 `samplerControl` 当前有 `Sampler` label 和当前 sampler value，但没有说明该 picker 用于选择这些生成参数的采样算法。
- `ParameterEditor` 同时用于 Generate 和 Prompt Template Editor，因此本轮继续选择共享参数编辑器中的小步 VoiceOver 语义优化，不修改 sampler 列表、picker 绑定或生成参数行为。

## 背景

Local Diffusion 的 sampler 是生成参数的一部分，会随 Generate 使用，也会随 Prompt Template Editor 中的参数一起保存。当前 picker 会朗读 `Sampler` 和当前 raw value，但 VoiceOver 用户聚焦控件时没有额外 hint 说明它会选择当前参数集的采样算法。

本轮只给 sampler picker 增加通用操作 hint。picker 标题、选项、tag、binding、sampler raw value、参数归一化、生成请求映射和模板保存/加载行为保持不变。

## 目标

- 在 `LocalDiffusion/Views/Shared/ParameterEditor.swift` 的 `samplerControl` 上补充：
  - `.accessibilityHint("Choose the sampling algorithm for these generation parameters.")`
- 保持现有 `Picker("Sampler", selection: samplerBinding)`、`ForEach(Sampler.allCases)`、`Text(sampler.rawValue).tag(sampler.rawValue)`、tint、accessibility label 和 accessibility value 不变。
- 语义文案必须通用，不能写死 Generate screen，因为该组件也用于 Prompt Template Editor。
- 同步 README、flow、flowchart、update_log，记录共享 sampler picker 获得采样算法选择范围的 VoiceOver hint。

## 非目标

- 不修改 `Sampler.allCases`、sampler raw value、显示文案、排序或默认值。
- 不修改 `samplerBinding` getter/setter 或 `GenerationParameters.normalizedForGeneration`。
- 不修改 native request mapping、stable-diffusion.cpp bridge、SwiftData schema、文件存储或 ViewModel。
- 不修改 Reset Defaults、Randomize Seed、Canvas Size preset、Width/Height、Steps、CFG 或 Seed 控件。
- 不修改 Generate、Prompt Library 的保存/加载模板行为。
- 不修改 Xcode project、CI workflow、Release asset、Mac Catalyst、StoreKit、购买、恢复、receipt、subscription、entitlement persistence 或付费门禁。
- 不新增 simulator VoiceOver 自动化、截图基线或真机 GGUF 生成验收。

## 当前架构依据

- `ParameterEditor` 属于 `LocalDiffusion/Views/Shared/ParameterEditor.swift` 的共享 UI 展示层，只负责展示和用户输入。
- Generate 和 Prompt Template Editor 通过 `@Binding var parameters: GenerationParameters` 复用同一个参数编辑器。
- `samplerControl` 当前只在 UI 层通过 `samplerBinding` 读写 `parameters.samplerRawValue`，不触碰 SwiftData、文件存储、ViewModel 或 native backend。
- 辅助功能 hint 属于 UI 表达，不应进入 ViewModel、数据层或 native 边界。
