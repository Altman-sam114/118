# v1.47 / 共享 CFG 控件语义

## 版本号

v1.47

## 版本分配依据

- 当前体验优化阶段已完成到 v1.46。
- v1.42 至 v1.46 已连续补充共享 `ParameterEditor` 中 Reset Defaults、Randomize Seed、Canvas Size preset、Sampler picker 和 Steps stepper 的 VoiceOver 语义。
- `cfgControl` 当前有 `CFG Scale` label 和当前值，但没有说明该 slider 调整 prompt guidance strength。
- `ParameterEditor` 同时用于 Generate 和 Prompt Template Editor，因此本轮继续选择共享参数编辑器的小步 VoiceOver 语义优化，不修改 CFG 范围、步进、默认值或生成行为。

## 背景

Local Diffusion 的 CFG Scale 是生成参数的一部分，会随 Generate 使用，也会随 Prompt Template Editor 中的参数一起保存。当前 slider 会朗读当前数值，但 VoiceOver 用户聚焦控件时没有额外 hint 说明它调整提示词引导强度。

本轮只给 CFG slider 增加通用操作 hint。Slider 标题、绑定、范围、step、当前值格式、参数归一化、生成请求映射和模板保存/加载行为保持不变。

## 目标

- 在 `LocalDiffusion/Views/Shared/ParameterEditor.swift` 的 `cfgControl` 中给 `Slider` 补充：
  - `.accessibilityHint("Adjust prompt guidance strength for these generation parameters.")`
- 保持现有 `Slider(value: $parameters.cfgScale, in: ..., step: 0.5)`、tint、accessibility label 和 accessibility value 不变。
- 语义文案必须通用，不能写死 Generate screen，因为该组件也用于 Prompt Template Editor。
- 同步 README、flow、flowchart、update_log，记录共享 CFG slider 获得 prompt guidance strength 说明的 VoiceOver hint。

## 非目标

- 不修改 `GenerationParameters.minimumCFGScale`、`maximumCFGScale`、默认值、step 或 normalization。
- 不修改 CFG slider binding、标题、当前值格式、可视布局或 Section 顺序。
- 不修改 Steps、Seed、Randomize Seed、Canvas Size preset、Width/Height、Sampler 或 Reset Defaults。
- 不修改 Generate、Prompt Library 的保存/加载模板行为。
- 不修改 native request mapping、stable-diffusion.cpp bridge、SwiftData schema、文件存储、ViewModel、Xcode project、CI workflow 或 Release asset。
- 不启用 Mac Catalyst，不新增 StoreKit import、product ID、购买、恢复、receipt、subscription、entitlement persistence 或付费门禁。
- 不新增 simulator VoiceOver 自动化、截图基线或真机 GGUF 生成验收。

## 当前架构依据

- `ParameterEditor` 属于 `LocalDiffusion/Views/Shared/ParameterEditor.swift` 的共享 UI 展示层，只负责展示和用户输入。
- Generate 和 Prompt Template Editor 通过 `@Binding var parameters: GenerationParameters` 复用同一个参数编辑器。
- `cfgControl` 当前只在 UI 层读写 `parameters.cfgScale`，不触碰 SwiftData、文件存储、ViewModel 或 native backend。
- 辅助功能 hint 属于 UI 表达，不应进入 ViewModel、数据层或 native 边界。
