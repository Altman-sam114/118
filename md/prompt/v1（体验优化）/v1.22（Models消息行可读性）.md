# v1.22 / Models 消息行可读性

## 版本号

v1.22

## 版本分配依据

- 当前体验优化阶段已完成到 v1.21。
- v1.21 补齐了 Plan 说明行的可读性和 VoiceOver 语义。
- Models 页面中每个模型行的 `progress.message` 仍使用普通小号 warning `Label`，承载下载失败、删除失败或状态提示信息时缺少统一的最小行高、换行约束和显式 VoiceOver 语义。

## 背景

Models 页面已经完成了下载、导入、删除、未跟踪文件管理、行内控制按钮和 Dynamic Type 堆叠优化。`ModelRow` 中仍有一处状态消息展示：

```swift
if let message = progress.message, !message.isEmpty {
    Label(message, systemImage: "exclamationmark.triangle")
        .font(.caption)
        .foregroundStyle(.orange)
}
```

这类 message 可能是失败原因、文件处理问题或下载状态说明，用户需要能清楚读到。当前实现字体偏小，未显式允许纵向换行，也没有明确辅助功能 label/value/hint。本轮只优化这条行内消息的展示组件，不改变任何下载、导入、删除或持久化行为。

## 目标

- 新增私有 `ModelMessageRow` 或等价小组件。
- 用该组件替换 `ModelRow` 中的 `progress.message` 裸 `Label`。
- 保留原有 message 文本和 `exclamationmark.triangle` 图标语义。
- 消息行应：
  - 使用真实 `Label` 和 SF Symbol。
  - 使用项目既有 Sci-Fi 颜色，建议使用 `SciFiTheme.amber`，避免直接 `.orange`。
  - 文本可自然换行，避免 Dynamic Type 下被压缩或截断。
  - 具有 44pt 最小高度。
  - 提供明确 VoiceOver label/value/hint，例如 label 为 `Model status message`，value 为实际 message，hint 说明这是该模型的下载或存储状态说明。
- 保持 `ModelRow` 的 header、progress bar、footer、control buttons、download status、delete/import/reconcile 行为不变。
- 更新 README、`md/flow/flow.md`、`md/flow/flowchart.md`、`update_log.md`，记录 Models message row 可读性改善。

## 非目标

- 不修改 `HuggingFaceDownloadManager` 状态机、下载、暂停、恢复、取消逻辑。
- 不修改 `LocalModel`、SwiftData schema、文件存储、导入、删除或 reconcile 行为。
- 不修改模型详情页、Add Model 表单、未跟踪文件导入编辑器或 alert 文案。
- 不新增 toast、banner、全局错误中心或日志系统。
- 不修改 native backend、生成流程、StoreKit、Mac Catalyst、Xcode project 或 CI workflow。
- 不改变 Models 页面导航、toolbar、sheet、fileImporter 或 confirmation dialog 行为。

## 当前架构依据

- `ModelLibraryView` 属于 UI 展示层，只能展示模型、下载状态、用户操作和错误反馈。
- 下载状态和 message 来源仍由 `HuggingFaceDownloadManager` / `ModelDownloadProgress` 维护。
- 文件存储必须继续经 `AppFileStore` 和现有导入/删除路径。
- SwiftUI 可访问性要求：重要状态或错误信息应可读、可换行，并提供明确 label/value/hint；行高应满足 44pt 阅读/触控基线。

   - 建议参数：`message: String`。
   - 使用 `Label` 显示 `message` 和 `exclamationmark.triangle`。
   - 文本使用 `.fixedSize(horizontal: false, vertical: true)`。
   - 使用 `.font(.callout)` 或与现有行内说明相容的系统字体。
   - 使用 `SciFiTheme.amber` 作为图标/文字主色。
   - 行使用 `.frame(maxWidth: .infinity, minHeight: 44, alignment: .leading)`。

   - 加 `.accessibilityHint("Describes this model's download or storage status.")` 或等价说明。

4. 不改 `progress.message` 的来源、不改 `ModelDownloadProgress`、不改任何按钮 action 或状态判断。
