# v1.38 / Prompt 分类菜单指针悬停语义

## 版本号

v1.38

## 版本分配依据

- 当前体验优化阶段已完成到 v1.37。
- v1.34 已为 iPad regular sidebar rows 和共享 Sci-Fi 主/次按钮建立系统 pointer hover affordance 基线。
- v1.35 和 v1.37 已分别为 Gallery 图片图块与 Gallery filter rail rows 建立系统 pointer hover affordance。
- Prompt Library 的分类 header action menu 是 iPad 宽屏模板管理中的高频入口，当前已有真实 `Menu`、可访问文本 label 和 44pt 最小命中区，但还没有显式 hit shape、系统 pointer hover affordance 或分类上下文 VoiceOver 语义。
- 本轮选择一个小步 iPad/Mac readiness UI 目标：为 Prompt Library 分类操作菜单触发器补充系统 pointer hover affordance 和分类上下文辅助功能语义，不改变模板、分类、搜索、编辑、加载或删除行为。

## 背景

Prompt Library 已支持模板分类、分类 rename/clear、模板 edit/load、搜索空状态和 Dynamic Type 友好布局。v1.11 已让分类和模板行控制具备可访问文本 label 与 44pt 命中区，v1.28 已改善空状态语义。

当前 `PromptCategoryHeader.categoryActions` 使用 `Menu` 暴露 `Rename Category` 与 `Clear Category`，并通过 `.labelStyle(.iconOnly)` 保持紧凑视觉，同时保留 `Label("Category Actions", systemImage: "ellipsis.circle")` 给辅助技术。对 iPad trackpad/mouse 用户来说，这个 44pt 菜单触发器还缺少和 sidebar、Gallery、共享按钮一致的系统 hover feedback；对 VoiceOver 用户来说，触发器没有说明当前作用于哪个分类。

本轮只补触发器 affordance 和语义，不碰模板数据或分类 mutation。

## 目标

- 在 `LocalDiffusion/Views/Prompts/PromptLibraryView.swift` 的 `PromptCategoryHeader.categoryActions` 上补充：
  - 与 8pt 圆角视觉一致的 `.contentShape(RoundedRectangle(cornerRadius: 8))`。
  - 系统 `.hoverEffect(.highlight)`。
  - 带分类上下文的辅助功能语义，例如：
    - label：`"\(group.title) category actions"`
    - value：`"Menu"`
    - hint：`"Opens rename and clear actions for this category."`
- 保持现有 `Menu`、`Rename Category`、`Clear Category`、`.labelStyle(.iconOnly)`、`SciFiTheme.cyan`、`.frame(minWidth: 44, minHeight: 44)`、Dynamic Type header layout 和分类 rename/clear 行为不变。
- 同步 README、flow、flowchart、update_log，说明 Prompt Library category action menu 获得 iPad pointer hover affordance 和分类上下文 VoiceOver semantics baseline。

## 非目标

- 不修改模板查询、搜索、分组、排序、删除、编辑、加载或分类 rename/clear 业务行为。
- 不修改 `PromptTemplateRow`、模板卡片 hover、edit/load 按钮、`PromptTemplateEditor`、`ParameterEditor` 或共享 `SciFiSecondaryButtonStyle`。
- 不修改 SwiftData schema、文件存储、native backend、Release asset、CI workflow、Generate、Models、Gallery 或 Plan 业务行为。
- 不启用 Mac Catalyst，不修改 `SUPPORTED_PLATFORMS`、`SUPPORTS_MACCATALYST`、Xcode project、签名、分发或 scheme 设置。
- 不新增 StoreKit import、product ID、价格、购买、恢复、receipt、subscription、entitlement persistence 或付费门禁。
- 不新增 visible keyboard shortcut 文案、Mac-only UI、自定义 hover 动画或额外手势。

## 当前架构依据

- Prompt Library 属于 `LocalDiffusion/Views/Prompts/PromptLibraryView.swift` 的 UI 展示层，只负责展示、输入、导航和用户反馈。
- `PromptCategoryHeader` 是分类 header 的局部 view；外层 `PromptLibraryView` 负责模板 query、search、sheet、confirmation dialog 和 SwiftData mutation。
- `Menu` 触发器适合保留为系统控件；本轮只增强其 hit shape、pointer hover 和辅助功能上下文。
- SwiftUI `hoverEffect(.highlight)` 是系统 pointer affordance，适合 iPadOS pointer 场景；本轮不做自定义 pointer animation。
