# v1.37 / Gallery 筛选栏指针悬停基线

## 版本号

v1.37

## 版本分配依据

- 当前体验优化阶段已完成到 v1.36。
- v1.34 已为 iPad regular sidebar rows 和共享 Sci-Fi 主/次按钮建立系统 pointer hover affordance 基线。
- v1.35 已为 Gallery 图片图块建立系统 pointer hover affordance。
- Gallery 左侧 filter rail 是 iPad 宽屏 Gallery 浏览的核心选择入口，当前 `GalleryFilterRow` 已有 44pt 最小高度和 VoiceOver label/value/hint，但还没有显式 hit shape 或 hover affordance。
- 本轮选择一个小步 iPad/Mac readiness UI 目标：为 Gallery filter rail rows 补充系统 pointer hover affordance，不改变筛选、排序、导航或数据行为。

## 背景

Local Diffusion 的 Gallery 在 iPad Root detail 中使用左侧 filter rail + 图片网格 + detail navigation 的单层宽屏布局。v1.17 已经让 All Images、folder、tag 筛选行显示数量并改善 VoiceOver 语义，v1.35 已经让图片图块具备 pointer hover affordance。

当前 filter rail row 仍然只依赖 `List(selection:)`、`.tag(...)` 和 `.sciFiListRow()` 的列表行为。对 iPad trackpad/mouse 用户来说，筛选行没有和 sidebar rows、Gallery image tiles 一致的 hover feedback。本轮只补系统级指针反馈和命中形状，不引入自定义动画或新业务状态。

## 目标

- 在 `LocalDiffusion/Views/Gallery/GalleryView.swift` 的 `GalleryFilterRow` 上增加明确 hit shape 和系统 hover affordance：
  - 保持现有 `Label`、title/count 文本、SF Symbol、44pt 最小高度、Dynamic Type 行为、accessibility label/value/hint。
  - 建议在 `.frame(maxWidth: .infinity, minHeight: 44, alignment: .leading)` 之后增加 `.contentShape(RoundedRectangle(cornerRadius: 8))`。
  - 建议增加 `.hoverEffect(.highlight)`。
- 同步 README、flow、flowchart、update_log，说明 Gallery filter rail rows 获得 iPad pointer hover affordance baseline。

## 非目标

- 不修改 `List(selection:)`、`.tag(...)`、`GalleryFilter` enum、folder/tag filter 计算、排序、详情导航或 Gallery 状态。
- 不修改 `.sciFiListRow()` 共享样式，避免扩大到所有列表。
- 不修改 Gallery 图片图块 v1.35 hover 行为。
- 不启用 Mac Catalyst，不修改 `SUPPORTED_PLATFORMS`、`SUPPORTS_MACCATALYST`、Xcode project、签名、分发或 scheme 设置。
- 不新增 StoreKit import、product ID、价格、购买、恢复、receipt、subscription、entitlement persistence 或付费门禁。
- 不修改 SwiftData schema、文件存储、native backend、Release asset、CI workflow、Generate、Models、Prompts 或 Plan 业务行为。
- 不新增 visible keyboard shortcut 文案、Mac-only UI、自定义 hover 动画或额外手势。

## 当前架构依据

- Gallery 属于 `LocalDiffusion/Views/Gallery/GalleryView.swift` 的 UI 展示层，只负责展示、输入、导航和用户反馈。
- `GalleryFilterRow` 是筛选栏 row 的局部 view；外层 `List(selection:)` 和 `.tag(...)` 负责选择状态。
- SwiftUI `hoverEffect(.highlight)` 是系统 pointer affordance，适合 iPadOS pointer 场景；本轮不做自定义 pointer animation。
- Gallery 数据仍来自 SwiftData `GeneratedImage`、`GalleryFolder` 和 tag 聚合；本轮不能触碰数据层、文件层或推理层。
