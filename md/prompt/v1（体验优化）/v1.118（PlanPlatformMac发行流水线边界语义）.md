# v1.118 / Plan Platform Mac 发行流水线边界语义

## 版本号

- 版本：`v1.118`
- 阶段：`v1（体验优化）`

## 版本分配依据

- 当前最新正式版本为 `v1.117 / Plan Mac 签名发行边界语义`。
- 人工总目标要求持续优化 Mac/iPad/UI/设计/付费功能。
- 本轮继续推进 Mac 平台状态透明度，将 Platform Status 的 Mac support note 与 v1.116、v1.117 的发行边界语义对齐，只改静态文案和 VoiceOver hint。

## 背景

Plan 页面 `Platform Status` 中的 Mac support note 目前说明 Mac support planned、iPhone/iPad app 可用、Mac/Catalyst app 未启用，并列出 platform support、native backend slice、signing decisions 和 UI validation 前置条件。v1.116 与 v1.117 已分别明确没有 separate Mac binary、Catalyst entitlement set、desktop distribution channel、Mac signing profile、sandbox entitlement 和 notarization pipeline。本轮需要让 Platform Status note 也包含这些发行流水线边界，避免该摘要层弱化 Mac 发行状态。

## 目标

1. 只增强 `PlanView.platformStatusContent` 中 Mac support `PlanNoteRow` 的 `text`。
2. 同步增强同一 note 的 `accessibilityHint`。
3. 明确当前没有 separate Mac binary、Mac signing profile、sandbox entitlement 或 notarization pipeline。
4. 保持 Platform Status summary rows、note label、icon、color、排序、row renderer 和所有业务逻辑不变。

## 非目标

- 不启用 Mac Catalyst、separate Mac target、separate Mac binary、Catalyst entitlement set、desktop distribution channel、Mac signing profile、sandbox entitlement、notarization pipeline 或 release channel。
- 不修改 Plan overview、Current Build、Capability Matrix、Entitlement Rules、Availability、Mac Readiness、Platform Status summary rows 或 navigation。
- 不启用 StoreKit、purchase UI、restore flow、receipt validation、subscriptions、entitlement persistence、paid gate、feature flags 或 unlock logic。
- 不修改 SwiftData schema、native backend、模型下载、生成、Gallery、Prompt Library、Models、workflow 或测试规范。

## 当前架构依据

- `PlanView` 位于 `LocalDiffusion/Views/RootContentView.swift`，属于 SwiftUI UI 展示层。
- `platformStatusContent` 使用 `PlanStatusSummaryRow` 和 `PlanNoteRow` 表达当前 iPhone/iPad 可用、Mac Catalyst 未启用和 Mac support planned。
- `PlanNoteRow.text` 是用户可见说明；`accessibilityHint` 是 VoiceOver 进一步解释。
- 本轮只改静态展示文案和辅助功能语义，不触碰状态层、数据层、文件存储、native backend、StoreKit、Xcode project 或平台配置。

## 实现步骤

1. 阅读 `RootContentView.swift` 中 `platformStatusContent` 与 `PlanNoteRow`。
2. 只修改 `accessibilityLabel: "Mac support status"` 的 `PlanNoteRow`：
   - text 继续说明 Mac support planned、iPhone/iPad app 可用、Mac/Catalyst app 未启用。
   - text 补充当前没有 separate Mac binary、Mac signing profile、sandbox entitlement 或 notarization pipeline。
   - accessibilityHint 同步说明当前没有 separate Mac binary、Mac signing profile、sandbox entitlement、notarization pipeline 或 active Mac distribution path。
3. 不修改 Platform Status summary rows、note label、systemImage、iconColor、布局或其他 Plan rows。
4. 同步更新 README、`md/flow/flow.md`、`md/flow/flowchart.md` 和 `update_log.md`。
5. 检查 diff，确认只包含本轮相关源码、文档和本提示词。
