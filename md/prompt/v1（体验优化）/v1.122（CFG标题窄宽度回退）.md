# v1.122 / CFG 标题窄宽度回退

## 版本号

- 版本：`v1.122`
- 阶段：`v1（体验优化）`

## 版本分配依据

- 当前最新正式版本为 `v1.121 / 参数控件窄宽度回退`。
- 人工总目标要求持续优化 Mac/iPad/UI/设计/付费功能。
- v1.121 已让共享参数控件中的 Seed/Randomize Seed 和 Size preset 支持窄宽度回退；本轮继续收敛同一共享参数区内的 CFG header 横向压缩风险。

## 背景

`ParameterEditor.parameterHeader(title:value:)` 当前只由 `cfgControl` 调用，用来显示 `CFG` 标题和当前数值。它在 accessibility Dynamic Type 下已经使用纵向布局，但普通 Dynamic Type 下固定使用 `HStack { Text(title); Spacer(); Text(value) }`。虽然 `CFG` 和数值较短，但在 iPad Split View、Stage Manager、Form sheet 或未来本地化文案变长时，固定横排仍可能压缩。项目已在 Generate 外层布局、Seed/Size 参数控件中采用 `ViewThatFits(in: .horizontal)` 作为窄宽度回退模式，本轮将同一模式补齐到 CFG header。

## 目标

1. 只优化 `ParameterEditor.parameterHeader(title:value:)` 的普通 Dynamic Type 分支。
2. 使用 `ViewThatFits(in: .horizontal)` 优先显示现有横排 header，横向空间不足时回退纵向 header。
3. 保持 accessibility Dynamic Type 仍直接使用纵向布局。
4. 保持 CFG slider、`parameters.cfgScale` 绑定、数值格式、辅助功能 label/value/hint 和视觉颜色不变。
5. 让 Generate 与 Prompt Library 中复用的参数控件同时受益。

## 非目标

- 不修改 Seed、Randomize Seed、Size preset、steps、width、height、sampler 控件。
- 不修改 `GenerationParameters`、参数归一化、生成门禁、prompt/model/result/Gallery handoff、SwiftData、file storage、native backend 或 StoreKit。
- 不调整 `GenerationView` 布局、Prompt Library 布局、Xcode project、workflow 或测试规范。
- 不做 simulator / Stage Manager 截图目检；本轮用 Swift parse 和云端 build 覆盖语法与构建风险。

## 当前架构依据

- `ParameterEditor` 位于 `LocalDiffusion/Views/Shared/ParameterEditor.swift`，属于共享 SwiftUI UI 展示层。
- `GenerationView` 和 `PromptLibraryView` 都复用 `ParameterEditor(parameters:)`。
- `parameterHeader(title:value:)` 当前只被 `cfgControl` 调用。
- `cfgControl` 的 slider 已有独立 accessibility label/value/hint；本轮只改 header 布局选择，不改 slider 行为或语义。
- 本轮只改 UI layout，不触碰状态层、数据层、文件存储或 native backend。

## 实现步骤

1. 阅读 `ParameterEditor.cfgControl` 和 `parameterHeader(title:value:)`。
2. 保留 `dynamicTypeSize.isAccessibilitySize` 分支的现有 `VStack`。
3. 将普通 Dynamic Type 的固定 `HStack` 改为：

```swift
ViewThatFits(in: .horizontal) {
    HStack {
        Text(title)
        Spacer()
        Text(value)
            .foregroundStyle(SciFiTheme.secondaryText)
    }

    VStack(alignment: .leading, spacing: 2) {
        Text(title)
        Text(value)
            .foregroundStyle(SciFiTheme.secondaryText)
    }
}
```

4. 不修改 `cfgValueText`、`Slider`、slider accessibility 或其他 controls。
5. 同步更新 README、`md/flow/flow.md`、`md/flow/flowchart.md` 和 `update_log.md`，说明共享参数 CFG header 支持窄宽度回退。
6. 检查 diff，确认只包含本轮相关源码、文档和本提示词。
