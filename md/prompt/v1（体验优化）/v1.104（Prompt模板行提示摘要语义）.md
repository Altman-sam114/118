# v1.104 Prompt 模板行提示摘要语义

## 版本号

- 版本：`v1.104`
- 阶段：`v1（体验优化）`
- 任务名：Prompt 模板行提示摘要语义

## 版本分配依据

- 最新已完成版本为 `v1.103`，已覆盖 Prompt template name/category 和 category rename name 输入框的 VoiceOver 语义。
- 本轮继续在 Prompt Library 内做小步 UI/可访问性优化，增强模板列表扫描体验，递增为 `v1.104`。

## 背景

`PromptTemplateRow` 目前 row-level accessibility value 只组合 positive prompt、steps、sampler 和 canvas size：

```swift
"Prompt \(template.prompt). \(template.steps) steps. Sampler \(template.samplerRawValue). Size \(template.width) by \(template.height) pixels."
```

这能帮助 VoiceOver 用户扫描模板参数，但仍缺少两个重要上下文：

- saved negative prompt / negative mask 是否存在。
- template category 是具体分类还是 Uncategorized。

模板列表的视觉分组和 prompt 文案仍保留不变，本轮只补 row-level VoiceOver 摘要，避免改变保存、加载、搜索、分组或 SwiftData 行为。

## 目标

1. 扩展 `PromptTemplateRow.templateAccessibilityValue`，让 VoiceOver 摘要包含：
   - Positive prompt 摘要，空值时明确 `No positive prompt`。
   - Negative prompt 摘要，空值时明确 `No negative prompt`。
   - Category 摘要，空值时明确 `Uncategorized`。
   - 现有 steps、sampler、canvas size 摘要。
2. 保留 row-level `.accessibilityLabel(Text("Prompt template: ..."))` 和 `.accessibilityHint(...)`。
3. 保留 Edit / Load 按钮、metric pills、可见 row 文案、Dynamic Type 布局和 `onLoad` / `onEdit` 行为不变。

## 非目标

- 不修改 PromptTemplate SwiftData schema。
- 不修改模板保存、编辑、加载、删除、搜索、分类分组或排序行为。
- 不修改 `matchesSearch(_:)`、`generation.load(template:)`、`renameCategory(_:to:)` 或 `clearCategory(_:)`。
- 不改变 Prompt row 可见布局、lineLimit、按钮样式、图标、hover/pointer 行为或 44pt 命中区。
- 不引入删除确认、swipe action 重写、新导航、新 sheet 或新状态。
- 不触碰 Generate、Gallery、Models、Plan、Root、StoreKit、Mac Catalyst、native backend、Xcode project 或 workflow。

## 当前架构依据

- UI 层位于 `LocalDiffusion/Views/**`，本轮只改 SwiftUI 展示语义。
- Prompt Library 负责模板保存、分类、搜索、编辑、加载；SwiftData 模型仍由 `AppModels.swift` 提供。
- `PromptTemplateRow` 已是模板列表 row-level VoiceOver 摘要的集中位置，适合单文件小步改动。
- 测试策略按 `md/test/test.md`：Swift 源码变更需跑本地轻量检查和两条 Swift parse，并通过 `main` push 触发 GitHub Actions 结果包。

## 验收标准

- `PromptTemplateRow` row-level VoiceOver value 包含 positive prompt、negative prompt、category、steps、sampler 和 canvas size。
- 空 positive prompt、空 negative prompt、空 category 都有明确可听状态，不读成空字符串。
- Edit Template、Load Template、metric pills 和现有 row-level label/hint 继续存在。
- 可见 UI、模板保存/加载/搜索/分类/删除行为、SwiftData schema、StoreKit、Mac Catalyst、native backend、project 和 workflow 均不变。
- 本地轻量检查和 Swift parse 全部退出 0。
- `origin/main` 最新 commit 对应的 GitHub Actions run 完成，Agent C 下载并核对未加密 artifact，manifest 的 `branch`、`commitSha`、`runId`、`runAttempt` 与最新状态一致，JUnit/summary/logs 均通过。
